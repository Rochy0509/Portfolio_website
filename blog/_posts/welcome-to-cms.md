# Welcome!

This is a sample blog post created through the CMS. You can:

- **Edit** this post by clicking on it in the admin panel
- **Delete** this post and create your own
- **Add images** by uploading them in the editor
- **Format text** using the rich text editor

## How to Create Posts

1. Go to `https://kenneth-martinez.com/admin`
2. Log in with your GitHub account
3. Click "New Blog Posts"
4. Fill in the fields and write your content
5. Click "Publish"

Your post will be automatically committed to GitHub and appear on your blog!
